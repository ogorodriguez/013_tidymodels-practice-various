Introduction

This notebook will reproduce the case study available in the tidymodels website, called “A predictive modeling case study” using a version of the data set called Hotel Booking from a study conducted by Antonio, Almeida, and Nunes (2019)

In this study, all of the steps for creating a predictive model are used. This section is the last of the Getting Started part of the tidymodels website. The previous sections covered one step of the predictive model creation separately and in detail.

I will attempt to follow the example here since I have a hard time not seeing the whole process holistically.

The dataset show an imbalance since only 8.1% of our date are hotel stays that contain children. This can be tackled with functions within tidymodels such as recipes however, this will be used as-is.
